### Client Engagements

**Software Engineer** · UniteGPS - Parent Invite Automation System
Built a transactional email service integrating Brevo's REST API to automate parent-invitation delivery for a SaaS platform serving 50-70 school districts. Implemented per-district domain white-labeling (SPF/DKIM) and 6 REST endpoints for delivery, logging, and analytics - validated at 100% delivery accuracy across 150 test invitations.
`Django` `MySQL` `Celery` `Redis` `Brevo API`

**Data Analyst** · Aperture Space, Inc. - Soil Moisture Analysis
Built a geospatial pipeline streaming Landsat 8/9 imagery via the STAC API and rasterio windowed reads to compute NDMI per station without downloading full scenes, then validated results against two years of ground-truth soil moisture data across 120 Oklahoma Mesonet stations in an interactive Plotly Dash dashboard.
`Python` `rasterio` `Pandas` `Plotly` `AWS S3`
